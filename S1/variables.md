## Variables

*Dentro de todos los programas
estructurados tenemos siempre 
3 tipos de instrucciones que podemos 
dar:*

1. Entrada/Salida
2. Cálculo
3. Control

Ejemplo: 

**Entradas:**
```
    a = 10
    b = 20 
    c = 5
```

**Cálculos:**
```
    a = a + 3 = 10 + 3 = 13
    b = b + 4 - a = 20 + 4 - 13 = 11
    c = a + b + c = 13 + 11 + 5 = 29
    a = a + c = 13 + 29 = 42
    b = 4
    c = c + 3 - b + 2 = 29 + 3 - 4 + 2 = 30
```

**Salidas:**
```
    a = 42
    b = 4
    c = 30
```
